  • Honours Project

    For my Honours Project, I researched the Capacitated Vehicle Routing Problem with Time Windows (CVRPTW).

    The problem is an NP-Hard combinatorial optimisation problem: it is thought to be unsolvable as it is not possible to perform a brute force search to find the best solution because the search space is too large. However, the best solution can be at least approximated in a feasible amount of time using AI.

    The AI methodology I used is known as a Genetic Algorithm. During the project, I experimented with two existing algorithms and wrote my own based on my findings. The algorithm I wrote (FIGA) proved better than the other two.
